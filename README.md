# YouTube_subtitle
YouTubuの英語字幕をDeepLで日本語訳にしてくれる。

# Usage
自動翻訳がある動画で、
1. 文字起こしを開く→コピペでテキストファイルに
2. ``python3 youtube_subtitle.py ファイル名``
3. 出力ファイルをDeepLにコピペ
